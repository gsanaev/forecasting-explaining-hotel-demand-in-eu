🧭 Notebook 2 — Forecasting Hotel Demand in Europe (2015–2025)¶

This notebook builds on the cleaned and harmonized dataset (hotel_panel_clean.csv) to model and forecast monthly hotel nights across 26 EU countries using econometric (SARIMAX / ARIMAX) and machine learning (XGBoost, LightGBM) techniques.

Goals:

  • Predict hotel demand (nights_spent) for a 12-month horizon (2025–2026).
  • Compare model performance across countries.
  • Identify temporal and macroeconomic drivers of recovery.

🟩 7. Econometric Forecast: ARIMAX (per country)¶

  • Purpose: capture autoregressive + macroeconomic relationships
In [7]:
arimax_exog = ["gdp", "unemployment_rate", "turnover_index", 
               "hicp_index", "covid_cases"]

panel["yhat_arimax"] = np.nan
regions = panel["region"].unique()

for country in regions:
    subset = panel.loc[panel["region"] == country]
    y = subset["nights_spent"]
    X = subset[arimax_exog]

    if y.isna().sum() > len(y) * 0.5:
        continue

    try:
        model = SARIMAX(y, order=(1,1,1), exog=X,
                        enforce_stationarity=False,
                        enforce_invertibility=False)
        res = model.fit(disp=False)
        panel.loc[subset.index, "yhat_arimax"] = res.fittedvalues
    except Exception:
        continue

print("✅ ARIMAX forecasts complete.")

✅ ARIMAX forecasts complete.

🟩 8. Machine Learning Forecast: XGBoost¶

  • Purpose: flexible nonlinear model across countries
In [8]:
exog_cols = ["gdp", "unemployment_rate", "turnover_index",
             "hicp_index", "covid_cases", "policy_stringency"]

# Include time & lagged features
features = (
    exog_cols
    + ["m"]
    + [c for c in panel.columns if c.endswith(tuple([f"_lag{L}" for L in [1,2,3]]))]
)
features = list(dict.fromkeys(features))  # remove duplicates

# Train-validation split (example: 2024 validation)
train = panel[panel["month"] < "2024-01-01"].copy()
valid = panel[panel["month"] >= "2024-01-01"].copy()

cat_cols = ["region"]
num_cols = features

X_train = train[cat_cols + num_cols]
X_valid = valid[cat_cols + num_cols]
y_train = train["nights_spent"].values
y_valid = valid["nights_spent"].values

# Preprocessor
pre = ColumnTransformer([
    ("cat", OneHotEncoder(handle_unknown="ignore"), cat_cols),
    ("num", "passthrough", num_cols)
])

# Model
xgb = XGBRegressor(
    n_estimators=800, learning_rate=0.03, max_depth=6,
    subsample=0.9, colsample_bytree=0.9,
    random_state=42, n_jobs=-1
)

pipe = Pipeline([("pre", pre), ("model", xgb)])
pipe.fit(X_train, y_train)
valid["yhat_xgb"] = pipe.predict(X_valid)

print("[INFO] XGBoost model trained and validated.")

def rmse(y, yhat):
    """Compute RMSE safely with NaN handling."""
    mask = (~y.isna()) & (~yhat.isna())
    if mask.sum() == 0:
        return np.nan
    return np.sqrt(mean_squared_error(y[mask], yhat[mask]))

def mae(y, yhat):
    """Compute MAE safely with NaN handling."""
    mask = (~y.isna()) & (~yhat.isna())
    if mask.sum() == 0:
        return np.nan
    return mean_absolute_error(y[mask], yhat[mask])

# Prediction columns
pred_cols = [c for c in ["yhat_naive", "yhat_arimax", "yhat_xgb"] if c in panel.columns]
print(f"[INFO] Available prediction columns: {pred_cols}")

# Compute RMSE / MAE for each top country
eval_tbl = (
    panel.query("region in @top_regions")
    .groupby("region", group_keys=False)
    .apply(lambda d: pd.Series({
        **{f"RMSE_{m}": rmse(d["nights_spent"], d[m]) for m in pred_cols},
        **{f"MAE_{m}":  mae(d["nights_spent"], d[m]) for m in pred_cols},
    }))
    .reset_index()
)

# Display results
